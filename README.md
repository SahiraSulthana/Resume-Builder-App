<h2>Project Overview</h2>
<p><b>Resume Builder App</b> is a full-stack web application designed to simplify resume creation. It allows users to input personal details, skills, education, and experience, and instantly generate a clean, professional resume layout. It is built using <b>HTML, CSS, JavaScript, React.js, Java Servlets, JDBC, and MySQL.</b> The application also supports live preview, database storage, and multiple resume templates, making it user-friendly and efficient.</p>
<h2>Features</h2>
<ul>
  <li> Dynamic Resume Builder Form</li>
  <li>Real-Time Resume Preview</li>
  <li>Save Resume Data to Database</li>
  <li>Multiple Resume Templates (Modern, Classic, Minimal)</li>
  <li>Responsive Design (Mobile & Desktop)</li>
  <li>Structured Data Handling using JDBC</li>
  <li>Fast and Interactive UI</li>
</ul>
